xpdView

I helped develop a software package for real-time data analysis, named xpdView, for the XPD beamline at NSLS-II as a part of a team of college interns and researchers. It was written in Python and used PyQt and MatPlotLib libraries for graphing and the user interface. The software is used by beamline scientists at XPD beamline to view experimental trends in data as they occur.
